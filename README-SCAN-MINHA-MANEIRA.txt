VERSÃO: SCAN À MINHA MANEIRA

Fluxo:
1. Ler código do toner
2. A app preenche Equipamento + Cor + Data
3. A app pede os últimos 3 dígitos da série
4. Procura na lista de impressoras
5. Seleciona a localização automaticamente

Notas:
- Data é preenchida com a data atual.
- Localização é preenchida como "SERIE - ARMAZEM - LOCAL".
- Se o código for novo, a app aprende o código e depois pede os 3 dígitos.
